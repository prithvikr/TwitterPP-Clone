## Twitter Profile Page Clone
This project is a clone of the Twitter profile page, created using HTML, CSS (Tailwind CSS), and JavaScript. It aims to replicate the look and feel of the Twitter profile page, allowing users to view and interact with tweets and profile information.

# Features

- Responsive design that adapts to different screen sizes.
- Profile header with cover photo and profile picture.
- Profile information section with bio, location, website, and join date.
- Tweet feed with the ability to post new tweets and interact with existing tweets (like, retweet, reply).

# Technologies Used
- HTML
- CSS (Tailwind CSS)
- JavaScript
# How to Use

- Clone the repository: git clone https://github.com/prithvikr/TwitterPP-Clone.git
- Open the index.html file in your web browser.
- Explore the profile page and interact with tweets.
